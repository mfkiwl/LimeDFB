# rx_path_top

## Description
Top module for packaging IQ samples received from [lms7002](./../lms7002/) module into stream packets. 

**Functionality:**
- Pack IQ samples from s_axis_iqsmpls stream int Stream packets
- Store Stream packets into AXIS buffer.
